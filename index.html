Activation Maximization through Gradient Ascent

How insane is AI generated art? I really enjoyed this project. This technique can be used to identify what a neural network is trained to see, which can help to inform an engineer whether or not the neural network was trained to percieve the correct patterns in an image. In this example, I've repurposed this method to generate an image by using repurposing a Convolutional Neural Network trained for image classification for feature reconstruction, by choosing the Neural Network layers that to maximize. The image is fed into the feature maximization model and adjusts the pixel values with gradient ascent, maximizing the excitation of the layers. In this way, we can identify what features the original CNN was looking for in the image, but we can also tune some Deep Dream hyperparameters to achieve abstract and often psychedlic art. My experience exploring this major at UC San Diego includes a study of cognitive systems, neural networks, calculus, linear algebra, statistics, and computer science. Want to detect wildfires automatically by using cameras and previous videos of wildfires? At my current internship at the San Diego Supercomputer Center, we are building a solution to this very problem by developing Deep Neural Networks in PyTorch, with the intent to deploy them on edge devices at high altitude weather stations.

Internship: Detecting Wildfires using Mask R-CNN Pt.2

This is much better. The next step is to try an LSTM approach where our model can remember predictions of previous frames, instead of simply searching for smoke frame-by frame. To do so, I will start by developing a Faster R-CNN that will feed the outputs to an LSTM. The Faster R-CNN will perform the detection and create bounding boxes that outline the image we want to feed to the LSTM. The intuition is that feeding a sequence of images will reduce the false positives of our system and increase the precision by monitoring the development of the region of interest over time by including the relevant information of past predictions.

Internship: Detecting Wildfires using Mask R-CNN Pt.1

It's a start. You see that the smoke is consistently predicted when it is there, but there are obvious false positives: clouds. Another issue is that non-maximum supression (NMS) is not removing other predictions on the same smoke because the Intersection over Union (IoU) is smaller than the NMS threshold. I either have to adjust this threshold or handle Mask R-CNN outputs with post-processing to favor the largest of the overlapping bounding box and segmentations.

Arduino Microcontrollers for Restoration of a 1974 Toyota FJ-40

This is version of the base program I developed as a private contractor for restoring all windows and window functions of a 1974 Toyota FJ40. The code was developed using a custom made debugging shield for the UNO R3 Arduino board. This code programs the Arduino microcontroller to perform all the expected functions of a car window with a focus on the longevity of power relays. The window functionality includes the typical directional commands (up vs down), automatic rolling on quick press, and automatic killswitch from voltage delta or canceling press.

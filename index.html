Pruning ResNet Blocks

Some Work as a Machine Learning Intern at Saildrone Inc.

During my ML internship at Saildrone I focused specifically on accelerating inference on the ocean drones. By replacing the backbone of the RetinaNet (SSD model) with a ResNet-18 and pruning the ResNet blocks of that model by 50%, I was able to achieve a 3.5x inference time over the previous model. I utilized structured pruning at 50% by manipulating the weight matrices of the backbone modules in an Object Oriented fashion. This model can now run in LOW POWER MODE in deployment environments, which is crucial since the drones operate on solar power, with FASTER performance than the previous model at HIGH POWER modes on Nvidia AGX cards.

Deep Reinforcement Learning: OpenAI Gym

Deep Deterministic Policy Gradient for Continuous Control

In Reinforcement Learning, one of the three major branches of Machine Learning, an agent can learn from interacting with its environment; learning from experience closely resembles animal and human intelligence more than any other branch of Artificial Intelligence. The Deep Deterministic Policy Gradient (DDPG) is a Deep Reinforcement Learning architecture for solving problems with high-dimensionality state spaces and continuous action spaces. Using 4 Dense Neural Networks, Actor-Critic methods, Experience Replay, and Polyak Averaging of Target network parameters, the agent is able to learn the value of actions in a given state and perform an optimal policy. This is my Python implementation based upon the 2016 ICRL Conference Paper "Continuous Control with Deep Reinforcement Learning", using TensorFlow and NumPy. I apply my implementation of the DDPG architecture on the OpenAI Gym's BipedalWalker-v2 environment. After only 1,000 simulated episodes, the bipedal walker learns the optimal amount of torque to be applied to each of its 4 joints across the state space to walk across a non-trivial 2-dimensional environment.

Deep Dream:
Visualizing a Neural Network

What is the Deep Dream Algorithm?

As image recognition, object detection and other deep neural networks get more advanced, they suffer more and more from a lack of interpretability. Interpretability is a critical and often overlooked aspect of Machine Learning; how can we trust the models we build? The Deep Dream algorithm offers a technique for absolving the poor interpretability of neural networks, by visualizing into the notorious "black box" of these models.

Deep Dream uses Gradient Ascent to perform Activation Maximization, also known as Feature Visualization, in order to reveal the features that a given convolutional neural network, its layers, or even individual 'neurons', are looking for. By maximally activating the brain of the model, we get vivid visualizations of the learned representations.

This is much better. The next step is to try an LSTM approach where our model can remember predictions of previous frames, instead of simply searching for smoke frame-by frame. To do so, I will start by developing a Faster R-CNN that will feed the outputs to an LSTM. The Faster R-CNN will perform the detection and create bounding boxes that outline the image we want to feed to the LSTM. The intuition is that feeding a sequence of images will reduce the false positives of our system and increase the precision by monitoring the development of the region of interest over time by including the relevant information of past predictions.

Arduino Microcontrollers for Restoration of a 1974 Toyota FJ-40

This is version of the base program I developed as a private contractor for restoring all windows and window functions of a 1974 Toyota FJ40. The code was developed using a custom made debugging shield for the UNO R3 Arduino board. This code programs the Arduino microcontroller to perform all the expected functions of a car window with a focus on the longevity of power relays.
